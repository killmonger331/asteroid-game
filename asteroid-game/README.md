# Asteroid Game
A classic Asteroids-style arcade game built in Python using Pygame.  
Control a triangular ship, shoot incoming asteroids, and survive as they split into smaller fragments.

## Features
- Player-controlled ship with rotation, thrust, and shooting mechanics
- Projectile-based asteroid destruction
- Dynamic asteroid splitting with velocity rotation
- Circle-based collision detection
- Frame-rate independent movement
- Modular object-oriented architecture
- Event logging for game state and gameplay events

## Play the Game

### Requirements
- Python 3.10+
- pygame 2.5+

```bash
git clone git@github.com:killmonger331/asteroid-game.git
cd asteroid-game
uv venv
uv pip install -r requirements.txt
uv run python main.py
```
## Linux / macOS / WSL
```
git clone git@github.com:killmonger331/asteroid-game.git
cd asteroid-game
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python main.py
```
